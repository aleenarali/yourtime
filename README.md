
## Project setup
- Create a file `.env` in the main directory with the line 
```
APIKEY=<your OpenWeather API KEY>
```
- The project can be run using in device 
```
yarn 
yarn start
```
- build of android
```
expo build:android
```

[Documentation the API](https://openweathermap.org/appid)
